# Neuroscience Coding Tools Reference

Quick reference of key libraries organized by analysis type. See `awesome-neuroscience/` for the full curated list.

## Python Libraries

### Behavior & Experiments
| Tool | Use Case | Link |
|------|----------|------|
| **PsychoPy** | Stimulus presentation, experiment design | [GitHub](https://github.com/psychopy/psychopy) |
| **expyriment** | Timing-critical behavioral experiments | [GitHub](https://github.com/expyriment/expyriment) |

### Neural Signals (EEG/MEG)
| Tool | Use Case | Link |
|------|----------|------|
| **MNE-Python** | EEG/MEG processing and analysis | [GitHub](https://github.com/mne-tools/mne-python) |
| **Nitime** | Time series analysis | [GitHub](https://github.com/nipy/nitime) |

### Imaging (fMRI/MRI)
| Tool | Use Case | Link |
|------|----------|------|
| **Nilearn** | Machine learning on neuroimaging | [GitHub](https://github.com/nilearn/nilearn) |
| **NiBabel** | Read/write neuroimaging formats | [GitHub](https://github.com/nipy/nibabel) |
| **DIPY** | Diffusion MRI analysis | [GitHub](https://github.com/nipy/dipy) |

### Simulation & Modeling
| Tool | Use Case | Link |
|------|----------|------|
| **Brian2** | Spiking neural network simulation | [GitHub](https://github.com/brian-team/brian2) |
| **Nengo** | Large-scale brain models | [GitHub](https://github.com/nengo/nengo) |
| **NEURON** | Biophysical neuron modeling | [neuron.yale.edu](https://neuron.yale.edu) |

### Calcium Imaging (Not in awesome list - common tools)
| Tool | Use Case | Link |
|------|----------|------|
| **CaImAn** | Calcium imaging analysis pipeline | [GitHub](https://github.com/flatironinstitute/CaImAn) |
| **Suite2p** | Two-photon processing | [GitHub](https://github.com/MouseLand/suite2p) |
| **CNMF-E** | One-photon/miniscope analysis | [GitHub](https://github.com/zhoupc/CNMF_E) |

### Electrophysiology (Not in awesome list - common tools)
| Tool | Use Case | Link |
|------|----------|------|
| **Kilosort** | Spike sorting | [GitHub](https://github.com/MouseLand/Kilosort) |
| **Phy** | Manual curation of spike sorting | [GitHub](https://github.com/cortex-lab/phy) |
| **SpikeInterface** | Unified spike sorting framework | [GitHub](https://github.com/SpikeInterface/spikeinterface) |
