## Essential Tutorial (Start Here)

### Machine learning classifiers and fMRI: a tutorial overview
- **Authors:** Pereira F, Mitchell T, Botvinick M
- **Journal:** NeuroImage, 2009
- **Citations:** 2,113
- **Link:** https://www.sciencedirect.com/science/article/pii/S1053811908012263
- **Why read:** The foundational tutorial paper. Covers basic concepts, preprocessing, feature selection, and classifier evaluation for fMRI data.

## Review Papers

### Autism spectrum disorder studies using fMRI data and machine learning: a review
- **Authors:** Liu M, Li B, Hu D
- **Journal:** Frontiers in Neuroscience, 2021
- **Citations:** 104
- **Link:** https://www.frontiersin.org/articles/10.3389/fnins.2021.697870/full
- **Topics:** ASD classification, resting-state fMRI, deep learning approaches

### Perspectives on machine learning for classification of schizotypy using fMRI data
- **Authors:** Madsen KH, Krohne LG, Cai X, Wang Y
- **Journal:** Schizophrenia Bulletin, 2018
- **Citations:** 22
- **Link:** https://academic.oup.com/schizophreniabulletin/article/44/suppl_2/S480/4938816
- **Topics:** Schizophrenia, feature extraction, clinical applications

## Methods Papers

### Multivariate analysis of fMRI time series: classification and regression
- **Authors:** Formisano E, De Martino F, Valente G
- **Journal:** Magnetic Resonance Imaging, 2008
- **Citations:** 190
- **Link:** https://www.sciencedirect.com/science/article/pii/S0730725X08000775
- **Topics:** SVM, relevance vector machines, multivariate pattern analysis

### Classification of ASD based on fMRI data with deep learning
- **Authors:** Shao L, Fu C, You Y, Fu D
- **Journal:** Cognitive Neurodynamics, 2021
- **Citations:** 100
- **Link:** https://link.springer.com/article/10.1007/s11571-021-09683-0
- **Topics:** Graph convolutional networks (GCN), deep learning, ASD

### Machine learning classification combining hyper-network of fMRI in Alzheimer's
- **Authors:** Guo H, Zhang F, Chen J, Xu Y, Xiang J
- **Journal:** Frontiers in Neuroscience, 2017
- **Citations:** 69
- **Link:** https://www.frontiersin.org/journals/neuroscience/articles/10.3389/fnins.2017.00615/full
- **Topics:** Hyper-networks, multi-kernel SVM, Alzheimer's disease

### Topological properties of resting-state fMRI networks improve ML autism classification
- **Authors:** Kazeminejad A, Sotero RC
- **Journal:** Frontiers in Neuroscience, 2019
- **Citations:** 158
- **Link:** https://www.frontiersin.org/journals/neuroscience/articles/10.3389/fnins.2018.01018/full
- **Topics:** Graph theory, network topology, biomarkers

### A hybrid machine learning method for fusing fMRI and genetic data
- **Authors:** Yang H, Liu J, Sui J, Pearlson G
- **Journal:** Frontiers in Human Neuroscience, 2010
- **Citations:** 255
- **Link:** https://www.frontiersin.org/articles/10.3389/fnhum.2010.00192/full
- **Topics:** Multimodal fusion, genetics + fMRI, schizophrenia
