# Imaging Methods

Analysis methods for neural imaging techniques.

## Categories

- **Calcium Imaging**: One-photon, two-photon, miniscope
- **fMRI**: Preprocessing, GLM, connectivity, decoding
- **Optical**: Fiber photometry, optogenetics analysis
- **Histology**: Cell counting, registration, atlas alignment

## Common Pipelines

### Calcium Imaging
1. Motion correction (NoRMCorre, Suite2p)
2. Cell segmentation (CNMF-E, Suite2p, CaImAn)
3. Spike inference (OASIS, CASCADE)
4. Neural activity analysis

### fMRI
1. Preprocessing (fMRIPrep, FSL)
2. First-level analysis
3. Group-level statistics
4. Visualization
