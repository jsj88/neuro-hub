# Behavioral Methods

Analysis methods for behavioral neuroscience paradigms.

## Categories

- **Navigation**: T-maze, Morris water maze, Barnes maze, radial arm maze
- **Open Field**: Locomotion, anxiety-like behavior, exploration
- **Fear Conditioning**: Contextual, cued, extinction
- **Operant**: Lever pressing, nose poke, reinforcement schedules
- **Social**: Social preference, resident-intruder, tube test

## Adding Methods

For each method, document:
1. Paradigm description
2. Key behavioral measures
3. Analysis pipeline (preprocessing → feature extraction → statistics)
4. Code implementation or references
5. Source papers
